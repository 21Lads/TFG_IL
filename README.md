# TFG
Driving Simulation using Maximum Entropy IRL imitation learning

Main.ipynb guides through entire pipeline, but simulation only shows aggregate statistics. 
Hence it should be ran in Simulator.py independently if adequate files are saved in /results.

Instructions for Simulator.py

0. Make sure imports are downloaded, especially highway-env
1. Run class, "main" is included
2. New screen will pop-up with blue vehicle and a road
3. Keyboard actions:
    SPACE - resume/pause,
    'n' - advance one step,
    'p' - capture probe (frame) to see actions at one timestep,
    'q' - quit simulation.
